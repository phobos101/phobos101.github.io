Check out some of my personal projects

CoffeeCoder

CoffeeScript REPL and learning environment

CoffeeCoder was my final week long project at General Assembly. CoffeeCoder is way to learn and teach CoffeeScript with quick modular lessons that can be attempted in an online REPL and written by any registered member. CoffeeCoder utilises the full MEAN stack and was incredibly rewarding and fun to build! I also utilised Mocha with Chai to make the backend in a TDD fashion.

Tech Used:

Node.JS, Express, MongoDB, Angular.JS, Mocha, Chai, Bootstrap

Dwarven Defender

JavaScript web game

Dwarven Defender is a clicker type game inspired by titles such as 'Cookie Clicker' and 'Clicker Heroes'. The game can be played indefinitely, with the enemy HP and damage scaling exponentially as the player progresses. This was built purely with HTML5 and CSS3. Once the stage was set, I built all the logic and movement of the ‘enemy’ div with OOP JavaScript and jQuery.

Tech Used:

HTML5, CSS3, JavaScript, jQuery

Carboot

Trading app build on Rails

Carboot is an item trading system that allows users to exchange items in a one-to-one manner. Carboot uses a minimal, clean user interface for ease of use and takes inspiration from trading systems found in modern massively multiplayer online games. Carboot was the first project I made in Ruby on Rails and utilises the backend fully with multiple models. The frontend uses Foundation as its framework.

Tech Used:

Ruby on Rails, Passport, Paperclip, Foundation, PostgreSQL

Anansi

Event agnostic adaptation of HackJammer

Anansi was written and after the HackJammer project as an adaptation of it. Anansi allows a user to search for any event via the eventful crawler that I wrote that utilises asynchronous lookups. Once the results come back in, the Google Map plugin renders the pins with the event details at the event location. I also fixed up some issues in HackJammer such as controlled sending of requests to the Google Geocoder API.

Tech Used:

jQuery, Node.JS, Express, MongoDB, Google API, Async
